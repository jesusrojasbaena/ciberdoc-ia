# CiberDoc IA

Web pública de **CiberDoc IA**, consultoría que crea cursos de ciberseguridad informática desde cero usando inteligencia artificial como apoyo educativo. Landing comercial + demo navegable del curso "Ciberseguridad básica con IA para usuarios y pequeñas empresas".

Estado: **Fase 1 en construcción** — web estática (HTML5 + CSS3 + JS vanilla) desplegada en GitHub Pages, sin backend.

La fuente de verdad del proyecto es [`docs/especificacion-ciberdoc.md`](docs/especificacion-ciberdoc.md). Cualquier cambio de alcance o estructura se decide ahí antes de tocar código.
